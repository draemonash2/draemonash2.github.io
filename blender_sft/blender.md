## ショートカットキー

| 説明 | キー |
|:---|:---|
| トランスフォーム適用<br>（現在位置/姿勢/スケールのまま、位置ALL0/姿勢ALL0deg/スケールALL1.0にする） | `Ctrl + a` -> `All Transform` |
| 面の向き反転 | 編集モードで面を選択 -> `メッシュ` -> `ノーマル` -> `反転` |
| 3Dカーソルを原点に移動 | `Shift + s` |
| 移動ツール選択 | `Shift + space` -> `G` |
| 回転ツール選択 | `Shift + space` -> `R` |
| トランスフォームツール選択 | `Shift + space` -> `T` |
