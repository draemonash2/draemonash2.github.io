## Tips

- `claude -p "指示"`も実行可能（ヘッドレスモード）
- セッションで`Shift + Tab`を入力すると、次の3つのモードが順に切り替わる。
    - 通常モード
    - 自動編集モード (accept edits) 
    - プランモード (plan mode)
- スラッシュコマンド
    - `/init`：claudeプロジェクト環境構築
    - `/clear`：コンテキストクリア
    - `/compact`：コンテキスト圧縮
    - `/resume`：前回セッション復帰
    - `/remind`：回答/変更巻き戻し
    - `/copy`：回答コピー
- ショートカットキー
    - プロンプトをエディタで開く：`Ctrl + g`
    - モデル切替：`Alt + p`
    - 思考過程の表示：`Ctrl + o`
    - バックグラウンドエージェント実行：`Ctrl + b`
    - パーミッションモード切替：`Shift + Tab`
